端到端加密 · 仅在你的设备上解密 · 支持自托管 · 完全开源——所有源码托管在 <https://github.com/zerx-lab/zpass>。

## 模块

| 路径 | 说明 |
|---|---|
| [`desktop/`](./desktop) | 桌面客户端 — Wails 3（Go + React） |
| [`extension/`](./extension) | 浏览器扩展 — Chrome / Firefox 自动填充 |
| [`phone/`](./phone) | 移动端 — Expo React Native |
| [`website/`](./website) | 官方网站 — Astro 静态站点 |
